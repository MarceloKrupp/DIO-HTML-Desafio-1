Programas e ferramentas

Voltar ao menu

Ao iniciar o módulo, fomos apresentados ao Visual Studio Code que é uma IDE, utilizada para programar diferentes tipos de linguagens e também utilizada para HTML e CSS que são linguagens de marcação e estilização.

Já no VS Code instalamos o Live Server que simplificando permite o usuário realizar atualizações direto ao salvar o arquivo sem ter o trabalho de terque ir recarregar a página do projeto. Foi também demonstrado apresentado uma ferramente a qual já vem adicionada ao VS Code que sugere e completa com facilidade códigos que o usuário esteja escrevendo.

Tags

Voltar ao menu

HTML

Está tag define o documento HTML, tudo dentro da tag de abertura e fechamento HTML será lido como um arquivo HTML pelo navegador.

HEAD

Está tag define o cabeçalho da página, nela são definidos titulos, descrições e linkados arquivos por exemplo.

BODY

Está tag é o corpo do site, tudo que estiver dentro dela será mostrado para o usuário, imagens e textos por exemplo.

P

Está tag cria um parágrafo, então tudo que digitar dentro dela será organizado em um parágrafo.

Small

Está tag faz o texto que está dentro ser renderizado em um tamanho menor que o normal. Exemplo: Small

H1, H2, H3, H4, H5, H6

Estas tags são utilizadas para títulos, variam seu tamanho de fonte pré-programados de acordo com seu número, sendo h1 o maior e h6 o menor. Exemplo:

  1. H1

  2. H2

  3. H3

  4. H4

  5. H5
  6. H6

Mark

Está tag marca a palavra que estiver dentro com um fundo amarelo. Exemplo: Mark

Atributos de Textos

Voltar ao menu
Underline

Para utilizar uma linha embaixo da palavra utilizamos a tag "u" que ela colocará um underline nas palavras que estiverem dentro dela. Exemplo: Underline

Itálico

Para deixar a palabra "deitada" mais conhecido como Itálico, você precisa utilizar a tag "i" que ela deixará as palavras que estiverem dentro dela em itálico. Exemplo: Itálico

Strong

Para deixar a palavra em negrito, você precisa utilizar a tag "Strong" que ela deixará as palavras que estiverem dentro dela em Negrito. Exemplo: Strong
